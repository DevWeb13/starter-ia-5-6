# Prompt de revue indépendante

```text
Travaille strictement en lecture seule. Ne modifie aucun fichier, Git, GitHub ou service externe.

Compare le résultat au brief, puis cherche erreurs, contradictions, risques, jargon, répétitions et complexité inutile.

Classe chaque constat : bloquant s’il interdit un usage sûr ou une réussite vérifiée ; important s’il doit être corrigé avant livraison ; amélioration s’il est facultatif. Donne pour chaque point une preuve précise et la correction minimale recommandée. Utilise QUALITY.md comme référence lorsqu’il est disponible.

Si aucun bloquant ou important n’existe, dis-le clairement. Ne valide aucune action que tu n’as pas pu vérifier sur sa cible réelle.
```
