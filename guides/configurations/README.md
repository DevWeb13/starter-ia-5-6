# Choisir une configuration Starter IA

Utilisez la configuration la plus simple qui couvre la mission. Pour une mission complète, préparez un brief avec [`../../templates/BRIEF.md`](../../templates/BRIEF.md) et suivez [`../../WORKFLOW.md`](../../WORKFLOW.md).

| Configuration | À choisir pour | Exécution |
|---|---|---|
| [Chat](chat.md) | réfléchir, décider, produire un brouillon court | conversation |
| [Work](work.md) | conduire une mission cloud complète avec fichiers et outils | environnement cloud |
| [Codex local](codex-local.md) | inspecter, modifier et tester un dépôt depuis VS Code | machine locale |
| [Codex Remote](codex-remote.md) | piloter depuis iPhone une session Codex restée sur la machine | machine locale pilotée à distance |
| [Work + Codex](hybrid-work-codex.md) | séparer préparation cloud et implémentation dans le dépôt | passage de relais explicite |

Ces guides décrivent le rôle et les limites des environnements. Ils ne garantissent pas leur disponibilité sur tous les comptes et ne remplacent pas leur aide intégrée ou la vérification des commandes installées.
